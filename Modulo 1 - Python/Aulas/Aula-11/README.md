# 🐼 Pandas

Pandas é uma das bibliotecas mais populares para análise e manipulação de dados em Python. Ela fornece estruturas de dados eficientes e ferramentas poderosas para trabalhar com tabelas, séries temporais e grandes conjuntos de dados.

## 📌 O que é o Pandas?

O Pandas é uma biblioteca de código aberto construída sobre o NumPy que facilita tarefas como:

- Leitura e escrita de arquivos (CSV, Excel, JSON, SQL etc.)
- Limpeza e transformação de dados
- Análise exploratória de dados
- Filtragem e agregação de informações
- Manipulação de séries temporais
- Visualização integrada com bibliotecas como Matplotlib e Seaborn

## 🚀 Instalação

Instale utilizando o pip:

```bash
pip install pandas
```

## 📥 Importação

```
import pandas as pd
```
