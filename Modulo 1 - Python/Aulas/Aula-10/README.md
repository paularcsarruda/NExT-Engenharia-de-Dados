# 🔢 NumPy

NumPy (Numerical Python) é a principal biblioteca para computação numérica em Python. Ela fornece estruturas de dados eficientes para trabalhar com arrays multidimensionais e oferece funções otimizadas para operações matemáticas, estatísticas e científicas.

## 📌 O que é o NumPy?

O NumPy é a base de grande parte do ecossistema científico do Python, sendo utilizado por bibliotecas como Pandas, Matplotlib, SciPy e Scikit-Learn.

Com ele é possível:

- Trabalhar com arrays de alta performance
- Realizar operações matemáticas vetorizadas
- Manipular matrizes e tensores
- Gerar números aleatórios
- Executar cálculos estatísticos
- Desenvolver aplicações científicas e de Machine Learning

## 🚀 Instalação

Instale utilizando o pip:

```bash
pip install numpy
```

Ou utilizando o conda:

```bash
conda install numpy
```

## 📥 Importação

```python
import numpy as np
```
