## 📚 Sobre o curso

O programa NEXT da CESAR School tem como objetivo formar profissionais na área de tecnologia, com ênfase em Engenharia de Dados, abordando conceitos fundamentais e práticas modernas do ecossistema de dados.

## 🎯 Objetivo do repositório

Este repositório tem como finalidade:

- Organizar atividades e exercícios do curso
- Documentar a evolução ao longo dos módulos
- Armazenar projetos práticos
- Servir como material de estudo e referência

## 🛠️ Tecnologias utilizadas

- Python
- SQL
- Pandas
- Apache Spark
- Jupyter Notebook
- PostgreSQL
- Ferramentas de Cloud Computing
- Git e GitHub

## 🚀 Status do projeto

Em desenvolvimento contínuo durante o curso NExT.
