## 📚 Sobre o programa

O **NEXT**, promovido pela **CESAR School**, é um programa de formação voltado ao desenvolvimento de profissionais para o mercado de tecnologia.

Na trilha de **Engenharia de Dados**, o curso aborda conceitos, ferramentas e práticas utilizadas na coleta, organização, processamento, armazenamento e análise de dados.

Este repositório registra minha evolução ao longo da formação, reunindo exercícios, anotações e projetos práticos desenvolvidos durante os módulos.

---

## 🎯 Objetivos do repositório

* Organizar as atividades e os exercícios realizados durante o curso;
* Documentar os conteúdos estudados em cada módulo;
* Registrar minha evolução técnica em Engenharia de Dados;
* Desenvolver e armazenar projetos práticos;
* Criar um material de consulta e revisão;
* Praticar versionamento de código com Git e GitHub.

## 📈 Conteúdos estudados

Entre os principais temas abordados ao longo da formação estão:

* Fundamentos de Engenharia de Dados;
* Programação com Python;
* Manipulação e tratamento de dados;
* Bancos de dados relacionais;
* Modelagem de dados;
* Consultas SQL;
* Processamento de dados com Pandas;
* Introdução ao processamento distribuído;
* Apache Spark;
* Pipelines de dados;
* Computação em nuvem;
* Versionamento de código.
